Revisão de estudos

Uma aplicação em PHP que organiza os estudos em revisões periódicas

Abrir

O objetivo deste projeto foi permitir que eu pudesse adicionar qualquer estudo aprendido, para que a aplicação me dissesse quando deveria revisar o aprendizado.

Foi utilizado PHP tanto para fazer o back-end quanto para ser moldar os dados apresentados no front-end no padrão MVC. O banco de dados utilizado foi o MySQL com o mysqli.

Login simples com opção de registrar novo usuário. Cada usuário terá seu espaço de estudos, sem conexão entre eles.

A página inicial do aplicativo, onde o usuário pode verificar as revisões que fará no dia. Aqui ficam os estudos separados por área e por sub-área.

Gerenciar assuntos é responsável por mostrar todos os assuntos criados pelo usuário logado, onde podemos ver claramente os assuntos desativados, botões de editar e desativar assunto.

Tela de exibição de criação de novo assunto. Observação: a tela mostra uma seleção de áreas criadas antes de aplicar o restante do formulário. A partir desta página, o usuário pode criar áreas e sub-áreas como desejar.

Movedor de arquivos de pastas distintas

Pequeno app em Python para mover arquivos para certa pasta

Abrir

Esta aplicação foi desenvolvida para que qualquer arquivo do sistema fosse selecionado e movido para uma pasta desejada. A aplicação também cria um log para saber de onde veio cada arquivo.

Criei esta aplicação para que pudesse selecionar vários arquivos de diferentes pastas para fazer backup. Funciona tanto em distribuições Linux (testado no Ubuntu) quanto em Windows.

Ele permite selecionar vários arquivos de diferentes pastas e então selecionar o diretório que tais arquivos serão movidos para dentro. Observação: é possível resetar para limpar as adições.

Este arquivo é criado no diretório final, ele mostra de onde veio cada arquivo e onde foi definido o diretório final

Sorteador de Cartas Uno

Sorteia próxima carta a ser colocada no deck

Abrir

Sorteador simples em Javascript que diz a próxima carta a ser colocada no deck individualmente. O objetivo é utilizar quando o deck é novo e está em ordem.

Sabemos que Uno é facilmente manipulável se estiver em ordem, pois é um jogo que tende a ter uma sequência de cartas. Então quando comprado, talvez embaralhar normalmente faça as primeiras partidas um pouco ruins

Você clica em sortear e ele sorteia a próxima carta com o número e cor, ou coringa. O fundo fica com a cor da carta. E há um histórico de números e número de cartas restantes.

Controle de ponto

Controle de conto funcional para funcionários

Abrir

Projeto desenvolvido em aprendizado de PHP pela Cod3r, no padrão MVC, onde você pode utilizar a aplicação para controle de ponto dos funcionários da empresa.

É possível criar uma conta para o funcionário que pode registrar a hora de entrada, saída para almoço, entrada após o almoço e saída do dia.

Também é possível gerenciar as horas devidas do mês caso seja conta de gerente.

Login comum para entrar no sistema, onde cada funcionário utiliza. Como é um sistema de controle de ponto, a conta é criada por um gerente do sistema.

Esta tela é apresentada após o login e permite bater o ponto. Veja que o menu ao lado esquerdo é exibido em todas as páginas, onde podemos navegar e ver horas trabalhadas e hora de saída calculados. Outro detalhe é que mostra que este sistema calcula hora de almoço, por isso a divisão de batimento de pontos.

Página que todo funcionário tem para que acompanhe a situação das horas trabalhadas em todos os meses desde a criação do usuário. Ao final dela, é possível ver o tempo calculado das horas devidas do mês.ç

Esta tela permite verificar quem não bateu o ponto e algumas informações como número de funcionários e horas trabalhadas no total. É uma tela exclusiva do gerente.

Tela utilizada pelo gerente para criar ou editar informações das contas dos usuários cadastrados. Por segurança, apenas contas sem nenhum batimento de ponto podem ser excluídas, mas todas as contas comuns podem ser desativadas, na parte de editar, para evitar acesso a contas de funcionários desligados da empresa.
